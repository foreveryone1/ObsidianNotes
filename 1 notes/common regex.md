regex to select characters after brackets or after white space
```regex
(^|\[\[|\s)(\w)
```

lower case to upper case
```regex
\1\u\2
```

regex to select everything after the underscore
```regex
\_.*
```

regex to select top of file
```regex
^\A(?s)
```
